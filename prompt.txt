Chrome 확장 프로그램을 만들고 싶어. 유튜브에서 작동하고 다음 기능을 포함해줘:

1. 영상 재생 중 A를 누르면 현재 시간에 북마크를 추가하고, ±1초 내 기존 북마크가 있으면 삭제 후 새로 추가해.
2. S를 누르면 현재 시간에서 ±1초 이내 북마크를 삭제.
3. E를 누르면 현재 유튜브 영상의 모든 북마크를 삭제.
4. 북마크는 각 영상별로 저장되어야 하고, 영상 재시작 시 자동 불러와서 보여줘.
5. 우측 하단에 항상 북마크 리스트 오버레이가 보여야 하며, 각 항목을 클릭하면 해당 시간부터 다음 북마크까지 재생.
6. 현재 위치와 일치하는 북마크는 음영 처리해줘.
7. Q 키를 누르면 기능이 꺼졌다 켜졌다 하도록 토글되며, 이때 북마커 오버레이가 보이거나 숨겨지는 걸로 상태를 알 수 있어.
8. X 키를 누르면 가장 가까운 이전 북마커로 이동해서 다음 북마커까지 재생하고 정지.
9. Z 키를 누르면 X보다 더 앞의 북마커부터 현재 북마커 바로 전까지 재생하고 정지.
V: 영상 하단 25%만 가리되, 유튜브 컨트롤(재생바 등)은 그대로 보이게

이걸 기반으로 작동하는 Chrome Extension 전체 코드를 만들어줘.
